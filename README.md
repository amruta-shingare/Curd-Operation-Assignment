<h2> React & React Native CRUD App</h2>

**Overview**

This is a responsive CRUD application built with React.js (Web) and React Native (Mobile), allowing users to create, view, edit, and delete posts efficiently. The project demonstrates clean code practices, reusable components, and seamless UI interactions.

**Features**<br>
Create Post:- Modal form with mandatory fields (Title, Description, Image).<br>
View Posts:- Presentable, responsive layout.<br>
Update Post:- Edit via pre-filled modal form.<br>
Delete Post:- Smooth deletion with instant UI updates.<br><br>
**Tech Stack**<br>
React.js <br>
State Management:- useState, useEffect<br>
Storage:- Local Storage (Web) / Async Storage (Mobile)<br>
Styling:- Custom CSS (Web) / React Native Stylesheet<br>
UI Library:- Only for modal functionality<br>


**Run Project**<br>
npm install<br>
npm run dev
